## Installation

Copy skills to your clawd workspace:

```bash
git clone https://github.com/bolivinhoia-lab/clawd-skills.git
cp -r clawd-skills/* ~/clawd/skills/
```

Then restart your gateway:

```bash
openclaw gateway restart
```
